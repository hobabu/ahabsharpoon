# Ahab's Harpoon

Jekyll blog on GitHub Pages.

## Structure

```
├── _posts/          # Log entries
├── _layouts/        # Templates
├── assets/css/      # Stylesheet
├── calendar.html    # Economic calendar (Tradays)
├── index.html       # Log index
└── _config.yml      # Config
```

## New post

```
_posts/YYYY-MM-DD-title.md
```

## Local dev

```bash
bundle install
bundle exec jekyll serve
```
